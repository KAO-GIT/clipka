На схеме указаны названия кнопок клавиатуры, которые можно использовать при настройке горячих клавиш. Коды клавиш указываются в фигурных скобках {}. Например: {control alt ;}

Если на кнопке указано несколько названий - можно использовать любое из указанных. Регистр не имеет значения.

Символы {-}, {.} и {/} работают как для цифровой части клавиатуры, так и для обычной.

Названия на цифровой части, выделенные наклонным шрифтом, работают при включенном num_lock. Введенные обычным шрифтом работают при выключенном num_lock.

В Windows {control scroll_lock} выдает код для {control break} (https://en.wikipedia.org/wiki/Scroll_Lock)

Кроме того, имеется несколько специальных названий. Мне не хотелось использовать выражения REGEXP, хотя бы для того, чтобы не принимать во внимание регистр букв.
{any_space} – пробельные символы {space}, {tab},{ enter}
{letter} – буквы, {-letter} – не буквы
{digit} – цифры, {-digit} – не цифры

 

Описание горячих клавиш.

Может проверяться последовательность нажатия нескольких клавиш. Для этого названия клавиш должны быть указаны в поле ввода последовательно, например {b}{b}. 
Если нужно для задачи нужно использовать несколько разных вариантов горячих клавиш их можно указывать в разных строках, например
{b}
{r}

Кроме того, если в фигурных скобках указано несколько назаваний – они тоже правильно анализируются, например в случае {ctrl b r} будет срабатывать каждый вариант :
{ctrl b}
{ctrl r}
