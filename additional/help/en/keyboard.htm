The diagram shows the names of the keyboard buttons that can be used when setting hot keys. Key codes are indicated in figure brackets {}. For example: {control alt ;}

If the button indicates several names, you can use any of the indicated. The register does not matter..

{-}, {.} symbols work both for the digital part of the keyboard and for the usual.

The names on the digital keyboard, highlighted with inclined font, work when the num_lock is on. Else work with the num_lock turned off.

For Windows {control scroll_lock} equals {control break} (https://en.wikipedia.org/wiki/Scroll_Lock)

In addition, there are several special names. I did not want to use the REGEXP expressions, at least in order not to take into account the letters register..
